# Zombie Slayer 🧟

A retro arcade zombie shooter game built with Python, Pygame and Pyinstaller.

## Controls
| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| Spacebar | Shoot |

## Gameplay
- Survive waves of zombies
- You have **3 lives** — don't let them reach you!
- Get the highest score possible
